Insight from top 
The k-Nearest Neighbour (kNN) algorithm is a widely used classification method that predicts the class label of a query sample based on the characteristics of its nearest neighbors in the feature space [3]. Traditionally, kNN operates with a fixed k value, which can limit its effectiveness across diverse datasets [2]. To address this limitation, recent advancements propose a new kNN algorithm that learns an optimal k value for each test sample, taking into account the correlation among samples [2]. This approach enhances the algorithm's adaptability and accuracy in various tasks, including classification and regression. Moreover, an improved implementation of kNN has been developed, which incorporates local distribution and estimates class probabilities using Bayesian principles. This method assigns the query sample to the class with the highest posterior probability, thereby refining the classification process [4]. Additionally, a novel kNN model-based classifier has been introduced, which combines the strengths of both kNN and the Rocchio classifier, specifically tailored for text categorization challenges [5]. These innovations collectively enhance the robustness and applicability of kNN in contemporary machine learning tasks.
